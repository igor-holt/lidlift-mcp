Hi HN,

We just launched LidLift, an MCP tool that prevents AI agents 
from running the wrong tool.

The problem: AI agents often misuse tools. They clone repos 
when users want to view them, delete files when they mean to 
read them, and call wrong APIs.

LidLift solves this by analyzing tool-prompt fit before 
execution. It scores alignment, detects mismatches, and 
suggests safer alternatives—all in <100ms.

Try it live: https://mcp.iholt.workers.dev

Built for Claude/OpenCode MCP, runs on Cloudflare Workers,
