# LidLift - AI Vendor Partnership Package

## Partnership Overview

**LidLift** is the first MCP tool purpose-built for AI agent tool safety. We prevent costly misexecutions by scoring prompt-tool fit before agents invoke tools.

### Why Partner With LidLift?

**For AI Vendors:**
- ✅ Differentiate with built-in safety layer
- ✅ Reduce agent misexecution incidents
- ✅ Increase user trust and adoption
- ✅ Zero integration cost (MCP native)
- ✅ Live in <100ms at the edge

**For Users:**
- ✅ Prevent wrong tool execution (clone vs. view)
- ✅ Get safer alternative recommendations
- ✅ Reduce costly mistakes (accidental deletes, wrong clones)
- ✅ Real-time analysis with no API key needed

## Partnership FAQ

### Q: How does LidLift integrate?
**A:** MCP-native. One line of config for Claude/OpenCode. REST API for others.

### Q: What's the cost?
**A:** Free for public beta. Enterprise pricing TBD.

### Q: What's the latency?
**A:** <100ms p50, <500ms p99 at Cloudflare edge.

### Q: What data do you collect?
**A:** Anonymous usage metrics. No PII. GDPR compliant.

### Q: Can we white-label?
**A:** Yes, for Enterprise partners. Custom branding + private deployment.

### Q: What's the SLA?
**A:** 99.9% uptime for Enterprise. Best-effort for public beta.

### Q: How do you handle rate limiting?
**A:** Cloudflare rate limiting. 100 req/min for public beta. Unlimited for Enterprise.

### Q: Can we host it ourselves?
**A:** Yes, open source (Apache-2.0). Self-hosted option available.
