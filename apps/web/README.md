# LidLift Web

This app is the operator console for LidLift. It provides:

- the interactive prompt-to-tool analyzer
- the web API routes on `/api/analyze` and `/api/health`
- the launch surface intended for Vercel deployment

Start it with:

```bash
pnpm dev
```

See the root [README](/Users/igorholt/LidLift MCP/README.md:1) for the full stack, deployment flow, and environment setup.
